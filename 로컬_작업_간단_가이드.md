---

### 1️⃣ 프로젝트 빌드

```powershell
cd C:\WiseYoung_backend
.\gradlew clean build -x test
```

**목적:** 모든 코드가 정상적으로 컴파일되는지 확인

---

### 2️⃣ API 키 암호화

#### 암호화 키 생성
```powershell
java -cp "build/classes/java/main" com.example.youth.util.ApiKeyCryptoTool generate
```
**출력값 저장** → 이것이 `ENCRYPTION_KEY`

#### API 키 암호화
```powershell
java -cp "build/classes/java/main" com.example.youth.util.ApiKeyCryptoTool encrypt "AIzaSyC2SzM_GRKI0pr1AT4PkT7ID29hIM3hFhw" "위에서_생성한_암호화_키"
```
**출력값 저장** → 이것이 암호화된 `GEMINI_API_KEY`

---

### 3️⃣ 결과 저장

생성된 두 값을 안전하게 저장 (서버 배포 시 사용):

```
ENCRYPTION_KEY=생성된_암호화_키
GEMINI_API_KEY=암호화된_API_키
```
