## 📝 참고사항

- 현재 대부분의 API는 인증 없이 접근 가능 (개발 단계)
- 배포 시 JWT 토큰 기반 인증 필터 추가 필요
- SSL 인증서 설정 필요 (HTTPS)
- Flask 서버는 별도 구현 필요
- Gemini API 키 설정 필요
