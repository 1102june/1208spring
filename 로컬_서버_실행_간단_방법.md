# 🚀 로컬 서버 실행 간단 방법

## ⚡ 빠른 실행 (권장)

### 방법 1: PowerShell에서 직접 실행

PowerShell을 열고 다음 명령어를 **한 줄씩** 실행하세요:

```powershell
cd C:\WiseYoung_backend

$env:ENCRYPTION_KEY = 'qm9MfSTvjn8IvepsCn0cAqKXLxm+Nf6da40n0kMN2e4='
$env:GEMINI_API_KEY = 'iFfWIWG6SuICN30uyNLwZsh26/+DD9k7/tbBtCNSDycIve0IyNB2DrXSZTLWCC4B'

.\gradlew bootRun
```

**중요**: 작은따옴표(`'`)를 사용하세요! 큰따옴표(`"`)는 특수문자 때문에 문제가 발생할 수 있습니다.

---

### 방법 2: 스크립트 파일 실행 (수정됨)

```powershell
.\run-local.ps1
```

**만약 스크립트 실행이 안 되면**:

```powershell
# 실행 정책 확인
Get-ExecutionPolicy

# 필요시 실행 정책 변경 (관리자 권한 필요)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

---

### 방법 3: 수동 설정 후 실행

**터미널 1**: 환경 변수 설정
```powershell
cd C:\WiseYoung_backend
$env:ENCRYPTION_KEY = 'qm9MfSTvjn8IvepsCn0cAqKXLxm+Nf6da40n0kMN2e4='
$env:GEMINI_API_KEY = 'iFfWIWG6SuICN30uyNLwZsh26/+DD9k7/tbBtCNSDycIve0IyNB2DrXSZTLWCC4B'
```

**터미널 2**: 서버 실행
```powershell
cd C:\WiseYoung_backend
.\gradlew bootRun
```

## 🔍 환경 변수 확인

설정 후 확인:

```powershell
echo $env:ENCRYPTION_KEY
echo $env:GEMINI_API_KEY
```

값이 제대로 출력되면 설정 완료!

## ⚠️ 문제 해결

### "파일 이름, 디렉터리 이름 또는 볼륨 레이블 구문이 잘못되었습니다"

**원인**: 큰따옴표 사용 또는 특수문자 처리 문제

**해결**: 작은따옴표 사용

```powershell
# ❌ 잘못된 방법
$env:ENCRYPTION_KEY = "값+특수문자="

# ✅ 올바른 방법
$env:ENCRYPTION_KEY = '값+특수문자='
```

### 스크립트가 실행되지 않는 경우

```powershell
# PowerShell 실행 정책 확인
Get-ExecutionPolicy

# 정책이 Restricted이면 변경
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
