# 🚀 로컬 서버 실행 - 간단한 방법

## ✅ 가장 간단한 방법: 스크립트 사용

매번 환경 변수를 입력할 필요 없이, 한 번의 명령어로 실행:

```powershell
cd C:\WiseYoung_backend
.\run-local.ps1
```

이 스크립트가 자동으로:
1. 환경 변수 설정
2. 스프링 부트 서버 시작

### 방법 2: IntelliJ IDEA에서 환경 변수 설정

1. `Run` > `Edit Configurations...`
2. 실행 설정 선택 (예: `YouthApplication`)
3. `Environment variables` 필드에 추가:
   ```
   ENCRYPTION_KEY=TksJPfalBU2+pIU/A87y1nlswEHECggq5hS6JOOGZ2M=
   GEMINI_API_KEY=onZNUTFo6WhS02toy3wryzG2L8XVJiz+p1OJxbSmyeoNJv6ljk2hHRR4gfOr9RSX
   ```
4. `Apply` > `OK`
5. 이후 IDE에서 실행 버튼만 누르면 됨

**장점**: IDE에서 실행 버튼만 누르면 자동으로 환경 변수 적용

---

### 방법 3: 시스템 환경 변수로 설정 (영구)

Windows 시스템 환경 변수로 설정하면 매번 설정할 필요 없음:

1. `Win + Pause` > `고급 시스템 설정` > `환경 변수`
2. `시스템 변수` > `새로 만들기`
   - 변수 이름: `ENCRYPTION_KEY`
   - 변수 값: `TksJPfalBU2+pIU/A87y1nlswEHECggq5hS6JOOGZ2M=`
3. 다시 `새로 만들기`
   - 변수 이름: `GEMINI_API_KEY`
   - 변수 값: `onZNUTFo6WhS02toy3wryzG2L8XVJiz+p1OJxbSmyeoNJv6ljk2hHRR4gfOr9RSX`
4. `확인` 클릭
5. **IDE 재시작** (환경 변수 적용을 위해)

**장점**: 한 번 설정하면 영구적으로 사용 가능

**단점**: 다른 프로젝트와 충돌 가능성
