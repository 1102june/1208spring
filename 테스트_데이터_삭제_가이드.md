# 테스트 데이터 삭제 가이드

`housing_complex`, `housing_notice`, `policy` 테이블을 제외하고 나머지 테이블의 데이터를 모두 삭제합니다.

## 삭제되는 테이블

- `user` - 사용자 정보
- `user_profile` - 사용자 프로필
- `interest_category` - 관심사 카테고리
- `bookmark` - 북마크
- `calendar_event` - 캘린더 이벤트
- `user_activity` - 사용자 활동 로그
- `notification` - 알림
- `ai_recommendation` - AI 추천

## 유지되는 테이블

- `housing_complex` - 임대주택 단지 정보
- `housing_notice` - 임대주택 공고
- `policy` - 청년정책

## 실행 방법

### 방법 1: PowerShell 스크립트 실행 (권장)

```powershell
cd C:\WiseYoung_backend
.\clear_test_data.ps1
```

**주의**: MySQL/MariaDB 클라이언트가 설치되어 있어야 합니다.

### 방법 2: MySQL/MariaDB 클라이언트 직접 실행

```bash
mysql -h 127.0.0.1 -P 3306 -u root -p1234 wise_young < clear_test_data.sql
```

### 방법 3: MySQL Workbench 또는 HeidiSQL 사용

1. MySQL Workbench 또는 HeidiSQL을 실행합니다.
2. 데이터베이스 `wise_young`에 연결합니다.
3. `clear_test_data.sql` 파일을 열어서 실행합니다.

### 방법 4: Spring Boot 애플리케이션에서 실행

1. `clear_test_data.sql` 파일의 내용을 복사합니다.
2. 데이터베이스 클라이언트 도구에서 직접 실행합니다.

## SQL 스크립트 내용

```sql
-- 외래키 제약조건 비활성화
SET FOREIGN_KEY_CHECKS = 0;

-- 자식 테이블부터 삭제
DELETE FROM interest_category;
DELETE FROM bookmark;
DELETE FROM calendar_event;
DELETE FROM user_activity;
DELETE FROM notification;
DELETE FROM ai_recommendation;
DELETE FROM user_profile;
DELETE FROM `user`;

-- 외래키 제약조건 재활성화
SET FOREIGN_KEY_CHECKS = 1;
```
