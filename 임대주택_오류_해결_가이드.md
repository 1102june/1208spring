# 임대주택 데이터 조회 오류 해결 가이드

## 🔴 오류 메시지
```
"Query did not return a unique result: 2 results were returned"
```

## ✅ 해결 방법

### 1. 서버 재시작 필수
코드를 수정했지만 **서버를 재시작하지 않으면** 변경사항이 적용되지 않습니다.

```powershell
# 1. 현재 실행 중인 서버 중지 (Ctrl+C)

# 2. 서버 재시작
cd C:\Users\subpa\IdeaProjects\WiseYoung_backend
.\gradlew bootRun
```

### 2. 수정된 코드 확인

#### UserProfileRepository
- `findByUser_UserId` 메서드를 native query로 변경하여 LIMIT 1 적용
- 중복 결과 방지

#### UserRepository  
- `findByEmail` 메서드에 LIMIT 1 추가
- 중복 결과 방지

### 3. 데이터베이스 확인

만약 서버 재시작 후에도 오류가 발생한다면, 데이터베이스에 중복 데이터가 있을 수 있습니다.

#### 중복 UserProfile 확인
```sql
SELECT user_id, COUNT(*) as count 
FROM user_profile 
GROUP BY user_id 
HAVING COUNT(*) > 1;
```

#### 중복 User 확인
```sql
SELECT email, COUNT(*) as count 
FROM user 
GROUP BY email 
HAVING COUNT(*) > 1;
```

### 4. 문제가 계속되면

#### 서버 로그 확인
서버 콘솔에서 정확한 오류 메시지 확인:
- 어느 Repository에서 오류가 발생하는지
- 어떤 쿼리가 문제인지

#### 데이터 정리
중복 데이터가 있다면:
1. 가장 최신 데이터만 남기고 나머지 삭제
2. 또는 관리자 API를 통해 데이터 정리
