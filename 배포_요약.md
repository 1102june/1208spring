### ⚠️ 배포 전 필수 작업

#### 1. 안드로이드 앱 설정 변경
**파일**: `app/src/main/java/com/example/app/network/NetworkModule.kt`

```kotlin
// 변경 전
private const val BASE_URL = "http://127.0.0.1:8080/"

// 변경 후 (서버 IP로 변경)
private const val BASE_URL = "http://서버IP:8080/"
```

#### 2. 스프링 부트 설정 변경
**파일**: `src/main/resources/application.yml`

- [ ] `jpa.hibernate.ddl-auto`를 `validate`로 변경 (배포 환경에서만)
- [ ] 환경 변수로 민감 정보 관리 확인

#### 3. 환경 변수 설정
서버에서 다음 환경 변수 설정 필요:
- `DB_USERNAME`
- `DB_PASSWORD`
- `EMAIL_USERNAME`
- `EMAIL_PASSWORD`
- `LH_SERVICE_KEY`
- `GEMINI_API_KEY`
- `KAKAO_REST_API_KEY`

## 🆘 문제 발생 시

1. **서버 로그 확인**
   ```bash
   sudo journalctl -u wiseyoung-backend -f
   ```

2. **포트 확인**
   ```bash
   sudo netstat -tlnp | grep 8080
   ```

3. **데이터베이스 연결 확인**
   ```bash
   mysql -u wiseyoung -p wise_young
   ```

자세한 문제 해결 방법은 `KT_Cloud_배포_가이드.md`를 참조하세요.
